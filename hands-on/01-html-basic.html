Emmet で HTML を高速に書く

Emmet は HTML を省略記法で素早く書けるツールです。VSCode に標準搭載されています！

基本的な使い方

Emmet の記法を入力して Tab キーを押すと、HTML に展開されます。

🚀 よく使う Emmet 記法

Emmet 記法 展開結果
div + Tab <div></div>
div.container + Tab <div class="container"></div>
div#app + Tab <div id="app"></div>
ul>li*3 + Tab <ul>
  <li></li>
  <li></li>
  <li></li>
</ul>
table>tr*3>td*2 + Tab 3行2列のテーブルが一瞬で生成！
a[href="#"]{リンク} + Tab <a href="#">リンク</a>
! + Tab HTML5 の基本テンプレート

実践例：このページの要素を Emmet で書くと

例1: 見出しのセクション

通常の書き方:

<h1>タイトル</h1>
<h2>サブタイトル</h2>
<h3>小見出し</h3>

Emmet で書くと:

h1{タイトル}+h2{サブタイトル}+h3{小見出し} [Tab]

→ 一行で3つの見出しが生成されます！

例2: リストの作成

通常の書き方:

<ul>
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
  <li>Item 4</li>
  <li>Item 5</li>
</ul>

Emmet で書くと:

ul>li{Item $}*5 [Tab]

→ 一瞬で5つのリストアイテムが生成！（$ は連番）

例3: テーブルの作成

Emmet で書くと:

table>tr>th{見出し$}*3^^tr*2>td{データ$-$}*3 [Tab]

→ 見出し行 + 2行3列のテーブルが一瞬で完成！

例4: ナビゲーションメニュー

Emmet で書くと:

nav>ul.menu>li.menu-item*4>a[href="#"]{メニュー$} [Tab]

→ クラス名付きの4つのメニューが一発で生成！

Emmet 記法の記号まとめ

記号 意味 例
> 子要素（入れ子） div>p → div の中に p
+ 兄弟要素（並列） div+p → div と p を並べて
* 繰り返し li*5 → li を5つ
. クラス名 div.card → class="card"
# ID 名 div#app → id="app"
[] 属性 a[href="#"] → href 属性付き
{} テキスト p{Hello} → テキスト入り
$ 連番 li{Item $}*3 → Item 1, 2, 3
^ 親要素に戻る div>p>span^div → div 階層に戻る

📝 まとめ

今回学習した基本的なHTML要素：

  1. 見出し（h1〜h6）: コンテンツの階層構造を作る
  2. ul: 順序なしリスト（箇条書き）
  3. ol: 順序付きリスト（番号付き）
  4. table: データを表形式で表示
  5. hr: セクションを区切る横線
  6. div: 要素をグループ化するコンテナ
  7. a: リンクを作成
  8. Emmet: HTML を高速に書く省略記法

💪 演習問題

以下の課題に挑戦してみましょう！

課題1: 自己紹介ページを作成

以下の要素を使って自己紹介ページを作成してください：

  • h1 タグで名前
  • h2 タグで各セクション（趣味、スキル、経歴など）
  • ul で趣味のリスト
  • ol で経歴（時系列）
  • table でスキルレベル
  • hr でセクションを区切る
  • a タグでSNSやポートフォリオへのリンク

課題2: 好きなレシピを紹介

以下を含むレシピページを作成してください：

  • 料理名（h1）
  • 材料リスト（ul）
  • 作り方（ol）
  • 栄養成分表（table）

課題3: Emmet を使って高速コーディング 🚀

以下の構造を Emmet を使って作成してください：

  • table>tr>th{項目$}*4^^tr*3>td{データ$-$}*4 で4列3行のテーブル
  • nav>ul.menu>li.item*5>a[href="#"]{メニュー$} でナビゲーションメニュー
  • div.card>(h3.title{カード$}+p.description{説明文})*3 で3つのカード

ヒント: VSCode で記法を入力したら Tab キーを押すだけ！
