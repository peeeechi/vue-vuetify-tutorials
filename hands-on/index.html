🚀 Vue.js & Vuetify ハンズオン教材

Web 開発初心者向けの Vue.js と Vuetify のハンズオン教材です。

📚 学習コンテンツ

0. 環境構築

  • 📄 00-setup.md - VSCode の拡張機能のインストールと設定
  • 📖 00-emmet-practice.html - Emmet の練習ページ

1. 基本的な HTML 要素

📖 01-html-basic.html を開く

Web ページを作成する上で基本となる HTML 要素を学習します。

  • 見出し (h1〜h6)
  • リスト (ul, ol)
  • テーブル (table)
  • 水平線 (hr)
  • コンテナ (div)
  • リンク (a)
  • Emmet - HTML を10倍速く書く方法

2. JavaScript とイベント処理

📖 02-javascript-events.html を開く

JavaScript を使って HTML 要素にインタラクティブな機能を追加する方法を学びます。

  • DOM の操作 (getElementById, querySelector)
  • イベント処理 (addEventListener)
  • フォーム要素 (テキストボックス、チェックボックス、ラジオボタン、セレクトボックス)
  • 実践例 (文字数カウンター、簡単な計算機、フォームバリデーション)

3. Vue.js の基本

📖 03-vue-basic.html を開く

Vue.js を使って、簡単にインタラクティブな Web アプリケーションを作成する方法を学習します。

  • Vue.js とは？ (リアクティブなフレームワーク、Vanilla JS との比較)
  • CDN 方式でのセットアップ
  • v-model - 双方向データバインディング
  • v-bind / : - 属性のバインディング
  • イベントハンドリング (@click, @input, @change)
  • マスタッシュ構文 ({{ 変数名 }})

4. Vue.js の応用

📖 04-vue-advanced.html を開く

Vue.js の条件分岐、ループ、その他の便利な機能を学習します。

  • v-if / v-else-if / v-else - 条件分岐
  • v-show - 表示/非表示の切り替え
  • v-for - ループ処理
  • computed - 算出プロパティ
  • フィルタリング機能
  • 実践: To-Do リストアプリ

5. Vuetify の基本

📖 05-vuetify-basic.html を開く

Vuetify を使って、美しい Material Design の UI を簡単に構築する方法を学びます。

  • Vuetify とは？ (Material Design コンポーネントフレームワーク)
  • CDN 方式でのセットアップ
  • 主要コンポーネント (v-btn, v-text-field, v-card, v-list, v-chip, v-alert, v-form, v-data-table)
  • 実践例 (ユーザー登録フォーム、データテーブルでのユーザー管理)

💡 ヒント

デバッグ方法

  1. ブラウザの開発者ツール
    • Chrome: F12 キー
    • Firefox: F12 キー
    • Safari: Command + Option + I (Mac)
  2. console.log() を使う
  3. Vue Devtools を使う - Chrome/Firefox の拡張機能

よくあるエラー

  1. CDN が読み込まれていない - インターネット接続を確認
  2. Vue が動作しない - .mount('#app') の ID が正しいか確認
  3. イベントが動作しない - @click のスペルミスがないか確認
